## 对比

|    方式    |           优点           |   缺点   |
| :--------: | :----------------------: | :------: |
|    饿汉    |     线程安全、效率高     | 非懒加载 |
|    懒汉    |     线程安全、懒加载     |  效率低  |
|  双重检测  | 线程安全、懒加载、效率高 |    无    |
| 静态内部类 | 线程安全、懒加载、效率高 |    无    |
|    枚举    |     线程安全、效率高     | 非懒加载 |
